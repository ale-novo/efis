SimVimPanel must be run on a separate computer connected by LAN with the computer running X-Plane with SimVimX Plugin.

https://www.simvim.com

Download additional panel modules on the website.

-----------------------------------------------------------------------

Program hotkeys when panel module is loaded:

Arrow keys - Set screen position offset. Press "Enter" to save position. Hold "Shift" or "Alt" to move faster or slower.
B - Toggle panel background
F - Toggle fullscreen/windowed mode
M - Return to menu
Esc - Exit
